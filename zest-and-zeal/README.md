# Zest & Zest Website

This is a simple one-page website for the beauty parlour **Zest & Zest**.

## How to open the website

1. Open the folder `C:\Users\GAURAV\Documents\New project`
2. Double-click `index.html`
3. The website will open in your browser

## Files

- `index.html` contains the website content
- `style.css` contains the design and layout

## Easy edits later

If the client asks for changes, these are the most common things to edit inside `index.html`:

- Business name
- Phone number
- Address
- Services text
- Google Business Profile link

## Publish on GitHub Pages

1. Create a new repository on GitHub
2. Keep the repository public for free GitHub Pages hosting
3. Upload all files from this folder
4. In GitHub, open `Settings`
5. Open `Pages`
6. Under source, choose `Deploy from a branch`
7. Select branch `main`
8. Select folder `/ (root)`
9. Save

Your website link will look like:

`https://your-github-username.github.io/repository-name/`
